Data-center Systems Engineer at Cisco, based in Upper Nyack, NY. This site is a working index of things I've built on my own time — reproducible implementations of Cisco reference architectures + adjacent practitioner tooling, all runnable on free tier.

The premise: SEs shouldn't need a slide deck to answer "what does Cisco Secure AI Factory look like at scale?" — the answer should open in a browser tab. This site collects the artifacts that make that possible.

If you're a Cisco peer or a customer engineer curious about how the pieces fit together, hop into /code for the repos, /docs for the guides, or / for the architectural overview.
