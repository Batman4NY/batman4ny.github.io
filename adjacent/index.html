§ Adjacent tooling Standard DC operator tooling

Tools that don't map to a SAIF layer — but the lab needs them. Every DC operator has some version of these: out-of-band console access, emergency management-plane recovery, physical device provisioning. Not in the reference architecture, but assumed by everyone who's ever brought up a switch or firewall from a bare rack.

ConsolePi OOB · Serial console

Out-of-band (OOB) serial console server on Raspberry Pi hardware. Terminates serial from Cisco Secure Firewalls (1200 / 3100 / 4200) via USB-C native CDC-ACM, and from legacy DC gear via RJ45 with USB-serial adapters. Used for initial device bring-up and management-plane recovery.
